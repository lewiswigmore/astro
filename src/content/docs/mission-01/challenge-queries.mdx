### Challenge 1: The Security Breach
<span class="px-2 py-1 text-xs bg-green-900 text-green-200 border border-green-700 rounded font-mono">CLEARANCE: CADET</span>

**Objective:** Find all failed logins for the user "jadenman" and only show the timestamp and source IP.

> **Hint:** Use `where`, `==` (or `=~`), and `project`.

<details>
<summary className="cursor-pointer bg-space-800 border border-space-700 text-thrust-400 px-6 py-2 rounded hover:bg-space-700 transition-colors duration-200 font-mono text-sm inline-block mt-4">DECRYPT SOLUTION</summary>

<Holodeck initialCode={`AuthenticationEvents

| project timestamp, src_ip`} runFuelCost={5} client:load />
